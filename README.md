## Чтобы поднять приложение c базой с помощью docker необходимо выполнить:   
```bash
docker-compose build
```
```bash
docker-compose up
```

## При желании поменять порт, необходимо также его изменить в [Dockerfile](Dockerfile) и перебилдить `docker-compose`
